Overview
This project analyzes online sales data using SQL to understand monthly business performance. It calculates monthly revenue, order volume, and identifies the top 3 high-performing months using aggregation and grouping techniques.

 Dataset Information:
Table Name: online_sales
Column Name	Description
order_id	Unique ID for each order
order_date	Date of purchase
amount	Revenue generated from the order
product_id	Product identifier

Skills/Functions Used:
SQL Aggregations: SUM(), COUNT()
Date Extraction: EXTRACT()
Grouping & Sorting: GROUP BY, ORDER BY
Limit for top values: LIMIT

Key Queries Performed:
 Monthly Revenue
 Monthly Order Volume
 Top 3 Months by Sales

 Objective:
To evaluate monthly sales performance and identify patterns that help businesses optimize strategies, understand demand, and increase revenue growth.
